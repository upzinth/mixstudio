# MixStudio - Professional Music Services Web App

This is a modern, responsive web application for a premium music production service, built with React, Vite, and Tailwind CSS.

## Features

- **Premium Design**: Dark, luxurious aesthetic with Gold/Bronze accents.
- **Responsive UI**: Fully responsive navigation and layout for all devices.
- **Service Showcase**: Detailed detailing of services like Mixing, Mastering, and Production.
- **Portfolio Preview**: Interactive grid for showcasing past work.
- **Modern Animations**: Smooth transitions and hover effects.

## Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Fonts**: Montserrat (Headings), Inter (Body) - via Google Fonts

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, ServiceCard).
- `src/pages`: Page views (Home, About, Services, etc.).
- `src/layouts`: Layout wrappers (MainLayout).
- `src/assets`: Static assets.
